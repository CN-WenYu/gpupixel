## 简介

🚀 GPUPixel是一个实时、高性能图像和视频滤镜库，非常容易编译和集成，并且库文件非常小。

💻 它是使用C++11编写的，基于OpenGL/ES，并且带有内置的美颜效果滤镜，可以实现商业级别的效果。

🌐 它支持的平台包括`iOS`、`Android`、`Mac`、`Win`和`Linux`，理论上可以移植到任何支持OpenGL/ES的平台。
